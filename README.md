# Elite Client Onboarding Portal

A polished white, black, and gold React portal for guiding clients through project onboarding at Elite Era Development L.L.C.

## Features

- Welcome dashboard and onboarding progress
- Client and company information forms
- Project scope and confirmation controls
- Documents checklist with custom items
- Kickoff meeting details and availability slots
- Milestones and progress tracking
- Approval status and approval notes
- Communication log
- Local browser storage
- JSON import and export
- TXT export and print-friendly summary
- Responsive white, black, and `#f4af00` gold design

## Run locally

```bash
npm install
npm run dev
```
